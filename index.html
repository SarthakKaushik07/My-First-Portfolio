Projects

  1. Hostel Management System
    Class 12 Project · Python · File Handling (Binary / Text)

    A fully functional Python terminal application designed to automate hostel record management. The system uses a menu-driven interface and stores data in a text file (hostel_records.txt) acting as a lightweight database. Key features include adding student records (roll number, name, room number, fee status, date of birth, gender, phone number, year of study), viewing all students or filtering by year, searching by roll number, updating any field, and deleting records. Built-in duplicatecheck validation prevents erroneous entries. This project was my Class 12 Computer Science submission and significantly deepened my understanding of file handling, data integrity, and structured program flow in Python.

  2. Tic-Tac-Toe — Ultimate Edition
    Semester 1, VIT Vellore · Python

    A fully polished two-player Tic-Tac-Toe desktop game built using Python and the Pygame library. The game manages three distinct states — main menu, active gameplay, and game-over — through a clean state-machine architecture. The UI was built entirely from scratch: gradient backgrounds, hover-responsive buttons with shadow effects, and a semi-transparent modal overlay for results that auto-dismisses after 6.5 seconds. Win and draw detection covers all rows, columns, and diagonals. All visuals are drawn programmatically without image assets, and the game loop runs at a fixed 60 FPS using pygame.time.Clock.

  3. Weather Forecasting and Alerting System
    Semester 1, VIT Vellore · Python · API Integration

    A Python application that fetches real-time weather data via an external API and triggers automated alerts based on user-defined conditions such as rain warnings or temperature thresholds. This was my first project at VIT and introduced me to API integration, JSON data parsing, and automated notification logic — foundational skills that underpin most of my subsequent data-driven work.

  4. Waste Classifier (Collaborative Contribution)
    Collaborative · Google Teachable Machine · Machine Learning (Image Classification)

    A machine learning image-classification project built collaboratively to help a Class 10 student explore AI concepts independently. The model was trained using Google Teachable Machine to classify waste into three categories — Biodegradable, Recyclable, and Non-Recyclable — using webcam and uploaded image samples (~100–187 per category). This project offered hands-on exposure to the practical workflow of ML: data collection, training, testing, and model export (TensorFlow.js). It reinforced my understanding of how no-code/low-code ML tools can be used to prototype real-world solutions aligned with Sustainable Development Goal 12.

  5. Vital Sense — Miniature Sugar Test Machine
    Team Project · Inter-School Competition · IoT + Python + Face Recognition

    "Vital Sense" is an IoT-integrated miniature sugar-testing machine developed as a team project and presented at an inter-school competition. The device performs blood sugar level testing and is connected to the internet to transmit results directly to the patient's assigned doctor in real time. Additional features include a historical data viewer for tracking readings over time and a face recognition module that automatically identifies the patient and opens their personal health folder. Although the team did not claim a prize, the project received special recognition and appreciation from the judges and the principal of the host school — a meaningful acknowledgment of the concept's realworld impact potential.
